### SQLInjector

###### Description: 
Purpose of this program is to assist analysts in generating SQL scripts with ease. This allows you to insert contents from a .csv or .txt into the paratheses after an IN clause in the SQL statement. This came about due to the limiations of SQL Developer.

###### How it works:

- Step 1 Have a list of text or numbers to be inserted into the SQL statement created

![](https://i.imgur.com/JVE6pcG.png)

- Step 2 Run sql_injector_ver1.3.py with Python 2.7

![](https://i.imgur.com/E6QANBx.png)

- Step 3 Follow ordered steps in program then click generate button

![](https://i.imgur.com/x0J4kqy.png)

- Step 4 Output will be saved as SQL script

![](https://i.imgur.com/DFOaDSy.png)
